　 二、點字使用規則
　 (一) 使用規則應注意事項：

1. 國語點字中，有七個音 ( ㄓ、ㄔ、ㄕ、ㄖ、ㄗ、ㄘ、ㄙ ) ，其單獨成音時，為避免與英文二十六個字母中的 ( Ａ、Ｂ、Ⅰ、Ｇ、Ｈ、Ｊ、Ｅ ) 相混淆，點完其聲母後須加一個空韻母 ㄦ ( １，５，６ 點) 再注聲調，以示區別。

　 2. 國語拼音中，點字的結合韻不像一般國語注音符號的結合韻──以介母、韻母拼成，若有結合韻字，為了節省空間，減少篇幅，縮小點字圖書的體積，及便於摸讀，一律以一方表示，也就是利用結合韻的縮寫點寫，以一方一個符號表示。

例如：(1) 應用 (2) 國家
　
　 3. 在每一個國字拼音後，均須加注聲調記號。

　　如：陰平(第一聲) 為 ( ' ) 第三點。
　　　　陽平(第二聲) 為 ( 1 ) 第二點。
　　　　上聲(第三聲) 為 ( ` ) 第四點。
　　　　去聲(第四聲) 為 ( " ) 第五點。
　　　　輕聲 為 ( A ) 第一點。

4. 國語點字(含聲調)最多三方構成一個字，最少二方構成一個國字，且不論兩方或三方，字與聲調均應同一行點寫，不可分行點寫，若該行不夠點寫，則整個國字的拼音，均須移至下一行點寫。

5. 聲母中有三組點字記號相同。

　　如：(1) ㄍ 和 ㄐ 點字記號 為 ( k ) 1，3 點。
　　　　(2) ㄑ 和 ㄘ 點字記號 為 ( j ) 2，4，5 點。
　　　　(3) ㄒ 和 ㄙ 點字記號 為 ( E ) 1，5 點。

但是它們的拼音方法各不相同，使用時並不會互相混淆。
例如：
　 (1)公雞 ( ㄍㄨㄥ ㄐ一 )
　
　 ㄍ 音後接 (ㄨ) 音 為 (ㄍㄨ)；接 (ㄨㄥ) 音 為 (ㄍㄨㄥ)。
ㄐ 音後卻不可能接 (ㄨ) 音。

ㄐ 音後接 ( 一)音 為 (ㄐ一)。
ㄍ 音後卻不可能接 (一) 音 。

(2)其次 ( ㄑ一ˊ ㄘˋ )
ㄑ 音後接 (一ˊ) 音 為 (ㄑ一ˊ)；接(〡ㄥˊ)為 (ㄑ〡ㄥˊ)。
ㄘ 音後卻不可能接 (一) 音。
　
　 ㄘ 音後接 (ㄨ) 音 為 (ㄘㄨ) ；接(ㄨㄛ) 音 為 (ㄘㄨㄛ)。
ㄑ 音後卻不可能接 (ㄨ) 音 或 (ㄨㄛ) 音。

(3)習俗 ( ㄒ一ˊ ㄙㄨˊ )
　
　 ㄒ 音後接 (一ˊ) 音 為 (ㄒ一ˊ)；接( 〡ㄥˊ)為(ㄒ〡ㄥˊ)。
ㄙ 音後卻不可能接 (一) 音。

ㄙ 音後接 (ㄨˊ) 音 為 (ㄙㄨˊ)；接(ㄨㄛ) 音為(ㄙㄨㄛ)。
ㄒ 音後卻不可能接 (ㄨ) 音 或 (ㄨㄛ) 音。

因為ㄍ只和韻母ㄨ、ㄚ、ㄜ、ㄞ、ㄟ、ㄠ、ㄡ、ㄢ、ㄣ、ㄤ、ㄥ、ㄨㄚ、ㄨㄛ、ㄨㄞ、ㄨㄟ、ㄨㄢ、ㄨㄣ、ㄨㄤ、ㄨㄥ拼音。

而ㄐ只和韻母ㄧ、ㄩ、〡ㄚ、〡ㄝ、〡ㄠ、〡ㄡ、〡ㄢ、〡ㄣ、〡ㄤ、〡ㄥ、ㄩㄝ、ㄩㄢ、ㄩㄣ、ㄩㄥ 拼音。

ㄑ、ㄒ 只和ㄧ、ㄩ、〡ㄚ、〡ㄝ、〡ㄠ、〡ㄡ、〡ㄢ、〡ㄣ、〡ㄤ、〡ㄥ、ㄩㄝ、ㄩㄢ、ㄩㄣ、ㄩㄥ 拼音。

而ㄘ、ㄙ只和韻母ㄨ、ㄚ、ㄜ、ㄞ、ㄠ、ㄡ、ㄢ、ㄣ、ㄤ、ㄥ、ㄨㄚ、ㄨㄛ、ㄨㄞ、ㄨㄟ、ㄨㄢ、ㄨㄣ、ㄨㄥ拼音，所以並不會產生困擾。
　 (二) 通常國字注音符號的結構分為：

1.聲母 + 聲調
例如： 知恥、 知識、 次ㄖ、 自私
　
　 2. 聲母 + 韻母 + 聲調
例如：深根固柢、當仁不讓、生米熟飯、不勝枚舉。
　
　 3. 韻母 + 聲調
例如：異域、安逸、挨餓、遺物、恩愛、阿諛。
　
　 4. 聲母 + 結合韻 + 聲調
例如：中華民國、同病相憐、坐井觀天、天花亂墜、嬌小玲瓏。
　
　 5. 結合韻 + 聲調
例如：因為、原有、運用、牙牙、搖尾、雲煙。
